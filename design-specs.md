# Savora Recipe Cards: Design & Print Specifications

**TL;DR**
These cards are designed to bridge the gap between "Premium Chef Authority" and "Tuesday Night Practicality." They utilize high-contrast editorial typography and custom AI-generated product photography to guide the user through a 3-step cooking process.

## 1. Physical Specifications
To ensure the kits qualify for USPS Large Envelope rates while maintaining a premium feel, the recipe cards must adhere to the following:

| Spec | Value | Rationale |
| :--- | :--- | :--- |
| **Dimensions** | 5.5" x 8.5" (Half-Letter) | Fits perfectly in the 6" x 9" Savora Mailer. |
| **Paper Stock** | 16pt Matte Recycled Cover | Rigid enough to feel "premium," matte finish prevents glare in the kitchen. |
| **Coating** | Soft-Touch Aqueous (Optional) | Adds a "silky" texture that aligns with the sachet material. |
| **Finish** | Uncoated Back | Allows users to write notes or "date cooked" on the back if desired. |

## 2. Visual Hierarchy & Typography
The system uses the "Warm Minimalist" palette established in the Brand Identity.

- **Primary Font**: *Fraunces* (Editorial Serif). Used for titles and step numbers to signal artisanal quality.
- **Secondary Font**: *Inter* (Modern Sans). Used for instructions and ingredient lists for maximum legibility in low-light kitchen environments.
- **Color Logic**:
    - **Terracotta**: Hero text and branding.
    - **Saffron**: Step numbers (High visibility).
    - **Sage**: Category headers (Calming, herbal).
    - **Linen**: The background "Canvas."

## 3. The 3-Step "Simplified Gourmet" Framework
Every Savora recipe is strictly limited to 3 main instructional blocks. 
1. **The Bloom/Infuse**: Toasting spices to release oils.
2. **The Build**: Adding main proteins/veg and simmering.
3. **The Balance**: Final touches (sugar, acid, garnish) that make the dish restaurant-quality.

## 4. Printer Handoff Instructions
1. **Format**: Export the `recipe-cards.html` file to PDF using the "Print to PDF" function.
2. **Bleed**: The design includes a 0.125" safety margin. Ensure the printer uses a "Full Bleed" setup to avoid white borders on the hero images.
3. **Color Profile**: Ensure the printer matches the HEX values to **Pantone 7585 C (Terracotta)** and **Pantone 7527 C (Linen)**.
